# Prompt Buddy Starter Assets

These files back the starter pack that Prompt Buddy routes people into when it catches a weak ask.

The product direction is not "write prompts with a static checklist."
It is an observer-style companion that notices when the user is slipping and points to the smallest training file that fixes the mistake.

The current shipped slice is the observer engine, the terminal sidecar MVP, and these reusable training assets.
