# Prompt Buddy Starter

Prompt Buddy is meant to sit beside the user and intervene only when the prompt is about to waste a run.

This starter pack gives the observer something useful to point at:
- a before-send filter for generic prompt mistakes
- crypto-specific checklists for token DD, wallet review, and yield scans
- an intervention ladder for deciding how hard to interrupt
- a training map so the observer can route the user to the right fix fast
- weak-to-strong examples for when a rewrite is still the fastest move

The current shipped slice is the observer engine + training pack.
It is not pretending to be a full on-screen pet UI yet.
