# Training map

This is the routing layer Prompt Buddy should use when it notices a weak ask.

## Generic prompt mistakes
- vague task / weak success criteria → `checklists/before-send.md`
- missing output format → `checklists/before-send.md`
- missing evidence / verification → `checklists/before-send.md`
- missing environment / repo / chain context → `checklists/before-send.md`
- giant ambiguous ask → `intervention-ladder.md`
- asking for execution before exploration → `intervention-ladder.md`

## Crypto-specific routes
- token due diligence → `checklists/token-dd.md`
- wallet review → `checklists/wallet-review.md`
- yield scanning → `checklists/yield-scan.md`

## Observer rule
Do not dump the whole library.
Point to the one or two files that most directly fix the current mistake.
