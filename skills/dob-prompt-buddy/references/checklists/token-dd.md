# Token DD prompt checklist

A good token due-diligence prompt should include:
- exact token name, ticker, or contract
- chain if known
- risk tolerance or angle
- requested sections such as legitimacy, liquidity, concentration, red flags, and next checks
- a rule to separate facts from assumptions

Weak ask:
- "tell me if this token is good"

Stronger ask:
- "Review HYPE on Hyperliquid. Return: what it is, quick legitimacy check, liquidity questions, holder concentration risks, obvious red flags, and 3 next checks. Separate facts from assumptions."
